## Brief
"sanity" is a utility sanity checker provided by ICCAD'24 Contest Problem B. This checker checks the mapping connection and all other net and pin relative output integrity.
Currently, "sanity" cannot be executed on the contest virtual machine owing to the required library of GLIBCXX_3.4.26 or above.
Please kindly refer to other machines with GLIBCXX_3.4.26 or above installed.
## Usage 
$ ./sanity <testcase> <output>
## Example
$ ./sanity sampleCase sampleOutput
